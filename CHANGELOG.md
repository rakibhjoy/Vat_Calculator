## [1.0.0] - 2026-01-29

### Added
- Initial release of Bangladesh VAT Calculator
- Three calculation modes:
  - Add VAT to base price
  - Remove VAT from total price
  - Calculate VAT amount only
- Bangladesh-specific features:
  - BDT currency (à§³) support
  - Standard VAT rates (5%, 7.5%, 15%)
  - Bilingual interface (English & Bengali)
  - VAT Act 2012 compliance information
- Premium UI/UX:
  - Glass-morphism design
  - Animated gradient background
  - Smooth transitions and animations
  - Bangladesh flag icons
- Quick VAT rate buttons (5%, 7.5%, 10%, 15%, 20%, 25%)
- Detailed calculation breakdown
- Input validation with error messages
- Auto-formatting with comma separators
- Keyboard shortcuts (Enter key support)
- Responsive design for all devices
- Mobile-first approach
- Font Awesome icons integration
- Tailwind CSS styling
- Single-file application (no dependencies)

### Features
- **Currency Formatting**: Display amounts in BDT with proper formatting
- **Calculation Accuracy**: Precise calculations with 2 decimal places
- **Error Handling**: Comprehensive input validation
- **User Guidance**: Helpful hints and tooltips
- **Accessibility**: Semantic HTML and ARIA labels
- **Performance**: Fast loading and smooth animations
- **Cross-browser Support**: Works on Chrome, Firefox, Safari, Edge

### Documentation
- Comprehensive README.md
- Contributing guidelines (CONTRIBUTING.md)
- MIT License (LICENSE)
- Code of conduct
- Usage examples
- Deployment instructions

### Design
- Green gradient theme representing Bangladesh
- Clean, modern interface
- Professional typography (Poppins font)
- Intuitive user flow
- Visual feedback on interactions
- Premium shadow effects
- Smooth hover states

### Technical
